**Quick Overview**

create a directory called canada-food-guide inside the current folder.
Inside that directory, generate the initial project structure and install the transitive dependencies:

canada-food-guide
├── README.md
├── node_modules
├── package.json
├── .gitignore
├── public
│   ├── favicon.ico
│   ├── index.html
│   └── manifest.json
└── src
    ├── App.css
    ├── App.js
    ├── App.test.js
    ├── index.css
    ├── index.js
    ├── logo.svg
    └── serviceWorker.js
    └── setupTests.js
